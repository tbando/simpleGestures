### Overview (概要)

シンプルな動作とオプション設定を目指したGoogleChrome拡張機能のマウスジェスチャです。  
simpleGestures is GoogleChrome extension that adds a simple mouse gesture.


## How to build

1. clone source

```
git clone git@github.com:RyutaKojima/simpleGestures.git
```

2. package install

```
yarn
```

3. Build

```
yarn build
```

4. Check Style Guide

```
yarn lint
```
